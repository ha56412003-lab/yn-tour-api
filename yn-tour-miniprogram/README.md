# 云南旅游分销小程序（279机制）

> 浩哥要求本周完成的紧急任务 - 云南6天5晚跟团游分销小程序

---

## 项目概述

这是一个基于uni-app + Node.js + MongoDB的旅游分销小程序，实现了279分销机制、微信/支付宝支付、用户后台、管理后台等核心功能。

## 技术栈

- **前端**: uni-app (Vue 3 + TypeScript)
- **后端**: Node.js + Express
- **数据库**: MongoDB
- **支付**: 微信支付 + 支付宝

## 核心功能

### ✅ 已完成（后端）

1. **279分销机制**
   - 直推分润：30%/70%交替
   - 成团奖：800元/团（满7单）
   - 分红：平台营业额2%分红
   - 有效访问奖：0.5元/次（每日上限20元）
   - 锁客机制：用户下单后自动被上级锁客

2. **支付系统**
   - 微信支付集成
   - 支付宝支付集成
   - 模拟支付（用于测试）
   - 支付回调处理

3. **用户后台**
   - 收益统计
   - 收益明细
   - 提现功能
   - 我的团队
   - 订单管理

4. **管理后台**
   - 数据看板
   - 用户管理
   - 订单管理
   - 营收统计
   - 提现审核

## 快速开始

### 后端启动

1. 安装依赖
```bash
cd server
npm install
```

2. 配置环境变量
```bash
cp .env.example .env
# 编辑 .env 文件，配置数据库和支付信息
```

3. 启动MongoDB
```bash
# 确保本地MongoDB已启动，或使用MongoDB Atlas
```

4. 启动服务
```bash
npm run dev
# 或
npm start
```

服务将在 `http://localhost:3000` 启动

## 279分销规则

详细规则请查看：[279分销规则与分佣计算逻辑.md](./docs/279分销规则与分佣计算逻辑.md)

### 快速理解

1. **直推分润**：第1单30%，第2单70%，循环往复
2. **成团奖**：团队满7单奖励800元
3. **分红**：符合条件的用户分享平台营业额2%
4. **访问奖**：有效访问0.5元/次，每日上限20元
5. **提现**：最低100元，手续费1%
