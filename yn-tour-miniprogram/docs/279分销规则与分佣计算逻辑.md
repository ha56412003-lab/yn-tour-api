## 一、核心规则（最终确认）

1. **自购不算直推**，但算入团队总单量
2. **直推分润**：第1/3/5...单 30%（239.7元），第2/4/6...单 70%（559.3元），循环
3. **团队总单量** = 自购数量 + 直推数量
4. **成团**：每7单 = 1团，每团奖励800元
5. **分红池**：平台营业额 × 2%，按个人团队单量占比分配
6. **提现**：总收益 × 1% 手续费（最低1元），满100元可提

## 三、分佣计算

### 3.1 直推分润

```javascript
function calcDirectProfit(directPushNum) {
  const profitPerOrder1 = 799 * 0.3; // 239.7元
  const profitPerOrder2 = 799 * 0.7; // 559.3元
  const cycle = Math.floor(directPushNum / 2);
  const remainder = directPushNum % 2;
  return (cycle * (profitPerOrder1 + profitPerOrder2)) + (remainder * profitPerOrder1);
}
```

**示例**：直推12单 → 6个完整循环 + 0个剩余 = 6 × 799 = **4794元**

### 3.2 成团奖

```javascript
function calcGroupBonus(selfOrderNum, directPushNum) {
  const totalTeamOrder = selfOrderNum + directPushNum;
  const groupNum = Math.floor(totalTeamOrder / 7);
  return groupNum * 800;
}
```

**示例**：自购5 + 直推12 = 17单 → floor(17/7) = 2团 → 2 × 800 = **1600元**

### 3.3 分红

```javascript
function calcDividend(selfOrderNum, directPushNum, platformTotalRevenue, platformQualifiedTotalOrder) {
  const totalTeamOrder = selfOrderNum + directPushNum;
  const groupNum = Math.floor(totalTeamOrder / 7);
  // 分红资格：直推>=2 或 成团>=1
  const hasDividendRight = directPushNum >= 2 || groupNum >= 1;
  if (!hasDividendRight) return 0;
  const dividendPool = platformTotalRevenue * 0.02;
  return dividendPool * (totalTeamOrder / platformQualifiedTotalOrder);
}
```

**示例**：17单，分红池 = 50000 × 2% = 1000元，占比 17/100 → **170元**

### 3.4 总收益

```javascript
function calcTotalProfit(params) {
  const directProfit = calcDirectProfit(params.directPushNum);
  const groupBonus = calcGroupBonus(params.selfOrderNum, params.directPushNum);
  const dividend = calcDividend(params.selfOrderNum, params.directPushNum, params.platformTotalRevenue, params.platformQualifiedTotalOrder);
  return directProfit + groupBonus + dividend;
}
```

### 3.5 提现

```javascript
function calcFinalWithdrawAmount(totalProfit) {
  if (totalProfit < 100) {
    return { fee: 0, finalAmount: 0, note: '收益不足100元，暂不可提现' };
  }
  let fee = totalProfit * 0.01;
  fee = fee < 1 ? 1 : Math.round(fee * 100) / 100;
  return {
    fee: fee,
    finalAmount: Math.round((totalProfit - fee) * 100) / 100,
    note: '可正常提现'
  };
}
```

## 五、关键结论

1. **自购算团队量，不算直推分润**
2. **直推分润**：每2单为1个循环，30%+70%交替
3. **成团奖**：团队满7单成团，每团800元
4. **分红池**：平台营业额×2%，只有直推≥2或成团≥1才有资格参与分配
5. **提现门槛**：满100元，手续费1%（最低1元）
6. **所有模块统一使用此计算逻辑，不得另行定义**
