# 2026-04-18 项目经验-云南旅游分销小程序

## 一、问题处理记录

### 1. 公告弹窗 AnnouncementPopup 样式升级
- **问题**：弹窗样式粗糙，视觉层次不清
- **解决**：重新设计阴影层次、字体层次、按钮样式、蒙版模糊效果
- **文件**：`src/components/AnnouncementPopup.vue`

### 2. 图片资源超限（329KB → 140KB）
- **问题**：微信代码质量扫描提示图片超过200KB
- **解决**：用 `sips -s formatOptions 50` 压缩JPEG质量到50%
- **关键命令**：
  ```bash
  sips -s format jpeg -s formatOptions 50 --resampleHeight 300 input.jpg --out output.jpg
  ```
- **注意**：微信开发者工具可能缓存旧的扫描结果，需删除dist重新编译

### 3. JS压缩配置
- **问题**：代码质量扫描提示JS未压缩
- **配置检查项**：
  - `manifest.json` → `mp-weixin.setting.minified: true`
  - `vite.config.ts` → `minify: "terser"`
  - `project.config.json`（编译后生成）→ `minified: true`
- **结论**：配置已正确，编译后自动压缩，缓存导致误报

### 4. 组件按需加载 lazyCodeLoading
- **问题**：代码质量扫描提示未启用懒加载
- **解决**：
  - `manifest.json` → `mp-weixin.setting.lazyCodeLoading: "requiredComponents"`
  - `app.json`（编译后）会自动添加 `lazyCodeLoading` 字段
- **注意**：需删除dist重新编译才能更新app.json

### 5. 分享页"立即申请开通"按钮点击无反应
- **问题**：普通用户点击按钮没反应
- **根因**：导航路径问题 + 用户测试数据问题
- **解决**：
  - 确认导航路径 `/pages/distribution/join/join` 存在
  - 添加调试接口切换用户分销商状态
- **调试接口**：`POST /user/debug-toggle-distributor`
  ```json
  {"phone": "13900000001", "isDistributor": false}
  ```

### 6. 数据库直接操作（用户状态切换）
- **场景**：测试时需要把用户改成普通/分销商身份
- **命令**：
  ```javascript
  const user = await User.findOneAndUpdate(
    { phone: '13900000001' },
    { isDistributor: false, distributorLevel: 0 },
    { new: true }
  );
  ```

## 二、UniApp + Vue3 + 微信小程序关键配置

### manifest.json 关键配置
```json
{
  "mp-weixin": {
    "setting": {
      "urlCheck": false,
      "lazyCodeLoading": "requiredComponents",
      "minified": true,
      "postcss": true,
      "es6": true
    }
  }
}
```

### vite.config.ts 压缩配置
```typescript
export default defineConfig({
  plugins: [uni()],
  build: {
    minify: "terser",
    terserOptions: {
      compress: {
        drop_console: true,
        drop_debugger: true,
      },
    },
  },
});
```

### 图片压缩（macOS）
```bash
# 单张压缩
sips -s format jpeg -s formatOptions 50 --resampleHeight 300 input.jpg --out output.jpg

# 批量压缩
for f in *.jpg; do
  sips -s format jpeg -s formatOptions 50 --resampleHeight 300 "$f" --out "${f%.jpg}_new.jpg"
  mv "${f%.jpg}_new.jpg" "$f"
done
```

## 七、关键经验

1. **微信开发者工具缓存严重**：每次修改配置后删除dist重新编译
2. **图片压缩用sips**：macOS内置，无需安装额外工具
3. **用户状态测试**：用调试接口直接改数据库
4. **代码质量扫描**：配置正确的情况下，重新编译+清缓存即可通过
