### 今日完成：出行花絮评价功能（独立评价系统）

**后端新增：**
- `server/models/Review.js` — 评价数据模型（nickname, avatar, rating, content, image, show, createdAt）
- `server/routes/review.js` — 评价 CRUD 路由

  - POST `/api/review/delete` — 删除评价
  - GET `/api/review/list` — 管理端列表
  - GET `/api/review/public` — 公开列表（小程序用）
- `server/routes/withdraw.js` — 提现审核路由

**前端新增/改动：**
- `src/api/review.ts` — 评价 API 调用（独立于产品）
- `help.vue`（Tab2 出行花絮）：从 `/api/review/public` 读取评价，瀑布流展示
- 管理后台：`review-page` 独立评价管理页面
- 提现审核弹窗 ID 从 `review-modal` 改为 `withdraw-audit-modal`（避免 ID 冲突）
- 产品编辑弹窗中的评价区块已移除（两者独立）

## 下午/晚间更新：首页装修 + 素材管理 + 产品详情动态化

### 后端新增（第二轮）

**HomeConfig 模型：**
- `server/models/HomeConfig.js` — 首页装修配置模型（poster: 海报图URL, featuredProducts: 关联产品ID数组）
- `server/routes/homeConfig.js` — 首页配置路由
  - GET `/api/homeConfig/get` — 获取首页配置
  - POST `/api/homeConfig/set` — 设置首页配置

**Gallery 模型（素材管理）：**
- `server/models/Gallery.js` — 相册素材模型（title, images: [{url, type}], createdAt）
- `server/routes/gallery.js` — 素材管理路由
  - GET `/api/gallery/list` — 素材列表
  - POST `/api/gallery/create` — 创建素材
  - POST `/api/gallery/update` — 更新素材
  - POST `/api/gallery/delete` — 删除素材

**Review 模型已存在**，可直接使用

### 前端改动

- 动态读取 `/api/homeConfig/get` 获取海报图和推荐产品
- 海报图从 HomeConfig.poster 读取
- "查看详情"按钮从 HomeConfig.featuredProducts 动态渲染

**product.vue（产品详情页）：**
- `schedules` 字段：行程安排（数组，每项含时间+内容）

- `notes` 字段：预订须知（数组，每项含标题+内容）
- 均从 API 动态读取，不再 hardcoded
