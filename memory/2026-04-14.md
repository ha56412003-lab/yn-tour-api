### Jimp 1.x API 注意事项（踩坑记录）
- 导入: `const { Jimp } = require('jimp')` (不是 `const Jimp = require('jimp')`)
- 创建图片: `new Jimp({ width, height, color })` 不是 `new Jimp(width, height, color)`
- 缩放: `image.resize({ w, h })` 不是 `image.resize(width, height)`
- 写入: `poster.write(filepath)` (Jimp 1.x 同步)
- ⚠️ 注意：`circle()` 和 `radius()` 不是有效 Jimp 方法，已移除

## 修复：静态文件路径 + HTTPS 问题

### 静态文件路径修复
- 问题：后端返回 `/uploads/posters/xxx.png`，前端拼接 `BASE_URL`（含 `/api`）后变成 `/api/uploads/posters/xxx.png`，但静态文件原在 `/uploads`（无 `/api` 前缀）
- 修复：`index.js` 中 `app.use('/api/uploads', express.static(...))`
- 现在 URL 正确：`http://192.168.10.14:3000/api/uploads/posters/poster_xxx.png`

### HTTPS 问题
- 微信小程序要求图片必须用 HTTPS
- 开发环境是 HTTP，会有警告但可能仍能显示
- 生产环境需要给后端配置 HTTPS（nginx 反向代理或云服务）
